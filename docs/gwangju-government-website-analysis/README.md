# Gwangju Government Website Analysis

Research date: 2026-05-13 KST

This folder contains a UX, service-design, search, mobile, accessibility, technical, and AI-opportunity review of the official public websites for Gwangju Metropolitan City and five district/gu offices.

Websites reviewed:

- Gwangju Metropolitan City Hall: https://www.gwangju.go.kr/main.do
- Dong-gu Office: https://www.donggu.kr/index.es?sid=a1
- Seo-gu Office: https://www.seogu.gwangju.kr/kor/
- Nam-gu Office: https://www.namgu.gwangju.kr/
- Buk-gu Office: https://bukgu.gwangju.kr/
- Gwangsan-gu Office: https://www.gwangsan.go.kr/

## Key Findings

- The sites are mostly announcement-heavy and department/menu-heavy, while high-frequency citizen tasks require users to understand public-sector labels such as `전자민원`, `분야별정보`, `생활폐기물`, or `민원편람`.
- Search is essential, but it is still keyword/category search. It can find many official pages, but it does not summarize eligibility, fees, next steps, documents, deadlines, or district-specific rules.
- Gwangsan-gu currently has the strongest modernization and AI-positioning opportunity because it already exposes a prominent search-first layout, service cards, large-text controls, multilingual access, `광산 AI 이슈키워드`, and task pages for waste, welfare, parking, jobs, and civil services.
- Nam-gu's provided root URL opens a portal selector before the representative homepage, which creates avoidable entry friction.
- Buk-gu has an important reproducible issue: the public search/sitemap path for `대형폐기물 배출방법` led to a page-error URL during testing.
